# Visualización de Accidentes Viales - D3.js

## Descripción

Aplicación web interactiva desarrollada con D3.js para analizar accidentes viales. Permite identificar barrios con mayor accidentalidad, observar la evolución temporal, analizar personas heridas, vehículos involucrados y periodos con mayor concentración de accidentes.

## Dataset

- **Fuente:** Datos Abiertos Colombia
- **Tema:** Accidentes viales
- **Archivo utilizado:** `Accidentes_Viales_20260426.csv`
- **Descripción:** El dataset contiene registros de accidentes viales con variables como fecha, barrio, dirección, vehículos involucrados y personas heridas.

## Visualizaciones implementadas

1. Gráfico de barras horizontal para comparar barrios con más accidentes.
2. Gráfico de línea para analizar la evolución temporal.
3. Histograma para observar la distribución de heridos.
4. Gráfico de dispersión para relacionar vehículos involucrados y heridos.
5. Gráfico de dona para comparar accidentes con y sin heridos.
6. Mapa de calor para identificar accidentes por año y mes.
